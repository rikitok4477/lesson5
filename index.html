Первый селектор это селектор по тегу(допустим название элемента p,если мы записали его в css то ко всем параграфам будет применены свойства которые мы там напишем)

Второй селектор по id(он применятся только к конкретному элементу с заданным id)

Следующий селектор это селектор по классам

Он задаёт стилевой класс, который позволяет связать определённый элемент со стилевым оформлением. В значении допускается указывать сразу несколько классов, разделяя их между собой пробелом.

Следующий селектор это по двум классам(к любому элементу можно добавить несколько классов перечисляя иъ через пробел)
